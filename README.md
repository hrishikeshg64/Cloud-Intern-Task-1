# Static Website Hosting using AWS S3

## Project Overview
This project demonstrates how to host a static website using Amazon S3.  
The website is publicly accessible through a public URL after configuring static website hosting and proper bucket permissions.

## Technologies Used
- Amazon Web Services (AWS)
- Amazon S3
- HTML

## Project Steps
1. Created an S3 bucket in AWS.
2. Enabled Static Website Hosting.
3. Uploaded website file (HTML).
4. Configured bucket policy to allow public access.
5. Accessed the website through the public S3 endpoint.

## Website Files
- index.html

## Learning Outcome
Through this project, I learned how cloud storage services can be used to host static websites and how to configure permissions and bucket policies to make them publicly accessible.
